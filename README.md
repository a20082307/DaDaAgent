# DaDaAgent

## 使用步驟
1. git clone https://github.com/terrychou911019/DaDaAgent.git
2. cd DaDaAgent
3. git checkout -b "你的名字"(ex: git checkout -b terry)
4. 寫你的扣
5. git add *
6. git commit(仔細一點，至少要讓別人知道你寫了啥)
7. ***git push origin "你的名字"(ex: git push origin terry)***
8. 告訴大家你寫了一些東西
